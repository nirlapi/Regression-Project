## 📌 Project Overview
This project is a machine learning regression task developed for the Kaggle competition **[House Prices: Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)**. The goal of this project is to accurately predict the final sale price of homes in Ames, Iowa, using 79 explanatory variables describing almost every aspect of residential homes.

## 📊 Dataset
The project utilizes the standard Kaggle competition datasets:
* `train.csv`: Used for training and validating the model, includes the target variable (`SalePrice`).
* `test.csv`: Used for the final Kaggle submission, does not include the target variable.

## 🛠️ Data Preprocessing & EDA
We performed extensive Exploratory Data Analysis (EDA) and data cleaning to prepare the dataset for our machine learning models:
* **Initial Exploration:** Utilized statistical summaries (`describe()`), distribution analyses, and correlation matrices to understand feature relationships.
* **Automated EDA:** Integrated tools like `AutoViz` and `DTale` for rapid and comprehensive data visualization.
* **Handling Missing Values:** Based on our methodology, we established a strict **10% threshold** for missing data. Instead of imputing heavily missing data or dropping rows, we dropped columns with >10% missing values. 
  * *Dropped Features:* `LotFrontage`, `Alley`, `MasVnrType`, `FireplaceQu`, `PoolQC`, `Fence`, and `MiscFeature`.
* **Feature Engineering:** Utilized `Scikit-Learn` pipelines, `OneHotEncoder`, `OrdinalEncoder`, `MinMaxScaler`, and `StandardScaler`.
* *Note:* Principal Component Analysis (PCA) was intentionally omitted from this workflow with explicit approval from the course teaching assistant.

## 🤖 Modeling & Machine Learning
We trained and evaluated several advanced regression models to find the optimal predictions:
1. **Random Forest Regressor**
2. **K-Nearest Neighbors (KNN) Regressor**
3. **XGBoost Regressor**

### Final Model: Ensemble
To achieve the most robust and accurate predictions, we combined our top-performing algorithms into an **Ensemble Model** (using an averaging approach). The final predictions submitted to Kaggle are the calculated average of the predictions generated by these three models.

## 🚀 How to Run
1. Clone the repository and ensure the dataset files (`train.csv` and `test.csv`) are located in the same directory.
2. Install the required packages: `pip install pandas numpy scikit-learn xgboost matplotlib seaborn plotly autoviz dtale`
3. Run the Jupyter Notebook `ML4.ipynb` sequentially to see the EDA, preprocessing, model training, and prediction generation.
